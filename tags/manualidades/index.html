El concepto de pintura El término pintura procede del latín “pigmenta”, que se traduce por tinte o pigmento. Más allá de eso, el concepto de pintura puede hacer referencia a varias cosas: La pintura es el arte que usa pigmentos para hacer representaciones gráficas. El resultado de esas representaciones también se llama pinturas.
